# Systematic Review (PRISMA)

このスキルは、PRISMAガイドラインに基づいてシステマティックレビューを計画・実行するためのテンプレートです。

## PRISMAワークフロー

1. **研究質問の明確化**: PICO/PECO形式で定義します。
2. **検索戦略の設計**: データベース、キーワード、期間を定めます。
3. **文献検索**: 計画した検索式で包括的に検索します。
4. **スクリーニング**: タイトル/要旨→全文の順に除外基準を適用します。
5. **データ抽出**: 研究デザイン、アウトカム、バイアスなどを抽出します。
6. **品質評価**: CEBMなどでエビデンスレベルを評価します。
7. **統合と報告**: PRISMAフロー図とチェックリストに沿って報告します。

詳細なテンプレートは `resources/SEARCH_STRATEGY_TEMPLATE.md` と
`resources/PRISMA_CHECKLIST.md` を参照してください。
