# Evidence Grading (CEBM)

このスキルは、CEBM（Oxford Centre for Evidence-Based Medicine）のエビデンスレベルを用いて研究の質と信頼度を評価するためのガイドです。

## CEBMレベル表（概要）

| レベル | 代表的な研究デザイン | 例 |
| --- | --- | --- |
| 1a | システマティックレビュー（RCT） | 介入効果の統合解析 |
| 1b | 個別RCT | プラセボ対照試験 |
| 2a | コホート研究のシステマティックレビュー | リスク因子の比較 |
| 2b | 個別コホート研究 | 追跡調査 |
| 3a | 症例対照研究のシステマティックレビュー | リスク推定 |
| 3b | 個別症例対照研究 | 症例群と対照群の比較 |
| 4 | 症例集積・低品質研究 | 症例報告 |
| 5 | 専門家意見・機序推論 | 研究者の解釈 |

## 使用ガイドライン

1. 研究デザインを特定し、適切なCEBMレベルを割り当てます。
2. バイアス、サンプルサイズ、追跡期間などの品質指標を記録します。
3. 結論の強度を「エビデンスレベル」と整合させます。
4. 追加の詳細基準は `resources/CEBM_CRITERIA.md` を参照してください。
