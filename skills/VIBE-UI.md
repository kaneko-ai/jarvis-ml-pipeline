# VIBE-UI

## 前提
- 目的/対象/差別化を明言
- 美学トーンを一つに固定（例: brutalist, editorial, retro-futuristic）

## 要点
- タイポ: 決定的なフォント選択（汎用禁止）
- 色: 支配色+鋭い差し色
- 動き: ロード/スクロール/ホバーの一撃
- 構図: 非対称/重なり/グリッド破り

## 出力
- 1スクショ相当のモック or Storybook仕様
- 実装指針/トークン/コンポーネント命名
